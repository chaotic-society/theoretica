inline

Transform a vector by the matrix.

Multiplies the matrix by another matrix of the same type.

Parameters
BThe matrix to multiply with.
Returns
A new matrix containing the result of the multiplication.

This function performs matrix multiplication between the current matrix and another matrix B of the same type. The resulting matrix has the same number of rows as the current matrix and the same number of columns as B.

If the number of rows in B does not match the number of columns in the current matrix, an error is raised and an error matrix is returned.
